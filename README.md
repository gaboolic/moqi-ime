# Moqi IME - 输入法引擎

## 项目简介

Moqi IME 是一个用 **Go 语言** 开发的跨平台输入法引擎核心。

## 架构定位

```
┌─────────────────────────────────────┐
│         各平台 TSF/IM 层           │
│  (moqi-im-windows / macOS / Linux) │
├─────────────────────────────────────┤
│         Moqi IME 引擎层            │  ← 本项目 (Go)
│     (输入处理 / 候选生成 / 状态管理)  │
├─────────────────────────────────────┤
│         词库 / 语言模型层           │
└─────────────────────────────────────┘
```

## 核心职责

- **输入处理**：解析按键事件，管理输入状态
- **拼音解析**：拼音切分、音调处理、容错
- **候选生成**：基于词库生成候选词列表
- **用户词库**：学习用户输入习惯
- **跨平台抽象**：提供统一接口供各平台调用

## 支持的输入模式

本引擎支持两种输入模式：

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **拼音输入** | 标准汉语拼音输入，支持全拼/双拼、音调标记、智能纠错 | 普通用户日常输入 |
| **码表输入** | 支持自定义编码方案（五笔、郑码、仓颉等码表格式） | 专业输入用户、特定编码需求 |

两种模式共用同一套候选排序、用户词库学习等核心机制，可通过配置灵活切换。

## 与 Windows TSF 层的关系

`moqi-im-windows` 项目将实现 Windows 平台的 **Text Services Framework (TSF)** 适配层，通过 C 接口或 RPC 调用本引擎的核心功能。
